# EmImageWater

EmImageWater 是一个用于给图片添加水印的 Flutter 应用。

## 说明

此仓库包含 EmImageWater 的 Flutter 源代码，可在 Android、iOS、Windows、macOS、Linux 和 Web 平台上运行。

## 快速开始

1. 安装 Flutter 开发环境（https://docs.flutter.dev/get-started/install）。
2. 在项目目录运行依赖安装：

```bash
flutter pub get
```

3. 在连接设备或模拟器后运行：

```bash
flutter run
```

## GitHub 上传准备

建议的流程：

1. 在 GitHub 上新建仓库，例如 `EmImageWater`。
2. 本地初始化 git（如果尚未初始化）：

```bash
git init
git branch -M main
git add .
git commit -m "Initial commit: EmImageWater"
```

3. 将远程仓库关联并推送：

```bash
git remote add origin https://github.com/<your-username>/EmImageWater.git
```
